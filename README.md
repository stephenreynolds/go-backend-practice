# Go Backend Practice

Daily challenges and weekly projects to build Go backend skills.

## Structure

```
├── daily/YYYY-MM-DD/     # Daily challenges (~30-60 min)
├── weekly/week-NN/       # Weekly projects (ongoing)
├── CURRICULUM.md         # Learning roadmap
└── PROGRESS.md           # Progress tracker
```

## Goals

- Master Go fundamentals and idioms
- Build production-ready backend services
- Learn cloud-native patterns (Docker, K8s, microservices)
- Ship real projects

## Daily Rhythm

1. Morning: New challenge appears
2. Complete challenge (under 1 hour)
3. Commit solution with notes

## Getting Started

```bash
# Install Go (if needed)
# https://go.dev/doc/install

# Verify
go version

# Start today's challenge
cd daily/YYYY-MM-DD
cat README.md
```
